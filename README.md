# 自定义 excel 格式 导入导出

- JS 控制格式（单元格合并，样式）

## base

- 依赖 ![`xlsx`](https://www.npmjs.com/package/xlsx) ![`xlsx-style`](https://www.npmjs.com/package/xlsx-style) ![`file-saver`](https://www.npmjs.com/package/file-saver) 实现

- 支持导出格式： `csv` `xls` `xlsx` `json` `html`
